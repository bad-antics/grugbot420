# grug-group420 Projects

The grug-group420 ecosystem — simple developer tools with zero dependencies.

> *"complexity very very bad"* — [The Grug Brained Developer](https://grugbrain.dev/)

## Core Projects

### 🧠 [grugbot420](https://github.com/grug-group420/grugbot420)
Neuromorphic cognitive engine written in Julia. Models cognition through competing populations of pattern nodes with stochastic voting, domain partitioning, specimen persistence, and a biological immune system.
- **Language:** Julia
- **License:** MIT

### 📦 [Bindboss](https://github.com/grug-group420/Bindboss)
Self-extracting executable packer in Go. Pack any directory into a single portable binary with dependency checking, SHA-256 hashing, Ed25519 signing, and pre/post hooks.
- **Language:** Go
- **License:** MIT

### 🖥️ [grugbot-server](https://github.com/grug-group420/grugbot-server)
Web server for running grugbot420 with a browser-based interface. Bun.js powered.
- **Language:** JavaScript/TypeScript
- **Runtime:** Bun.js

### 🪨 [grug-cli](https://github.com/grug-group420/grug-cli)
Zero-dependency developer toolkit following grug-brained principles.
- **Language:** JavaScript
- **Runtime:** Bun.js

## Support Projects

### 🦴 [grug-install](https://github.com/grug-group420/grug-install)
One-click installers for Windows (PowerShell) and Linux/macOS (bash). Installs Bun, downloads grugbot-server, creates shortcuts.

### 🌐 [portal](https://github.com/grug-group420/portal)
Interactive web portal with live CLI playground. Hosted at [grug-group420.github.io/portal](https://grug-group420.github.io/portal).

### 📚 [docs](https://github.com/grug-group420/docs)
Documentation site. Hosted at [grug-group420.github.io/docs](https://grug-group420.github.io/docs).

### 📝 [grug-templates](https://github.com/grug-group420/grug-templates)
Zero-dependency starter templates for JavaScript, TypeScript, and HTML projects. No webpack, no babel, no config files.

### 📄 [grugbot420-paper](https://github.com/grug-group420/grugbot420-paper)
Research paper (LaTeX) documenting the grugbot420 architecture.
